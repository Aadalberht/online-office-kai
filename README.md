

Для того чтобы запустить проект «Онлайн-Офис» локально на MacBook, следуйте этой простой пошаговой инструкции:

Шаг 1. Экспорт исходного кода
В верхнем правом углу интерфейса Google AI Studio откройте меню настроек (иконка шестерёнки или три точки).

Выберите пункт Export (Экспортировать) и скачайте исходный код в виде ZIP-архива.

Распакуйте архив в удобную папку на вашем MacBook (например, на Рабочий стол или в папку Projects).

Шаг 2. Подготовка окружения (Установка Node.js)
Для работы проекта потребуется среда выполнения JavaScript — Node.js (версии 18 и выше).

Способ А (через официальный сайт): Скачайте и запустите установщик для macOS (рекомендуется версия LTS) с официального сайта nodejs.org.

Способ Б (через Homebrew в Терминале): Если у вас установлен пакетный менеджер Homebrew, выполните в Терминале:

code
Bash

brew install node

Шаг 3. Переход в папку проекта в Терминале
Откройте стандартное приложение Терминал (Terminal) на Mac (через Spotlight или из папки Программы -> Утилиты).

Введите команду перехода в папку проекта (cd) и укажите путь к распакованной папке:

code
Bash

cd /путь_к_вашей_папке/react-example

Подсказка: Вы можете написать cd , а затем просто перетащить папку с проектом из Finder прямо в окно Терминала — путь подставится автоматически.

Шаг 4. Установка зависимостей
Установите библиотеки, описанные в конфигурационном файле package.json (такие как React, Tailwind CSS, Lucide и Motion):

code
Bash
npm install
Установка займет около 10-30 секунд.

Шаг 5. Запуск сервера разработки
Запустите локальный веб-сервер Vite с помощью встроенного скрипта:

code
Bash
npm run dev
После успешного выполнения в Терминале появится сообщение с системными адресами:

code
Code
VITE v6.2.3  ready in 450 ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: http://192.168.1.XX:3000/
Шаг 6. Открытие приложения
Откройте любой веб-браузер на Mac (Safari, Chrome) и перейдите по адресу:
👉 http://localhost:3000

Система будет запущена со всеми локальными хранилищами, демонстрационными учётными записями, адаптивной версткой и интерактивным ИИ-интерфейсом!
